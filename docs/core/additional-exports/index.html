Additional Exports

make​

A collection of utility functions used to construct THREE objects like THREE.Vector*, THREE.Color etc... from different arguments

make(THREE.Vector2, 10); // new THREE.Vector2(10, 10);
make(THREE.Vector2, [10, 20]); // new THREE.Vector2(10, 20);
make(THREE.Quaternion, [10, 10, 10, 10]); // new THREE.Quaternion(10, 10, 10, 10);
make(THREE.Color, 'black'); // new THREE.Color('black');

is​

A collection of utility functions used to assert object type

is.ref(a); // checks if an object is a Ref
is.object3d(a); // checks if an object is a THREE.Object3D

is.camera(a); // checks if an object is a THREE.Camera
is.perspective(a); // checks if an object is a THREE.PerspectiveCamera
is.orthographic(a); // checks if an object is a THREE.OrthographicCamera

// and more
