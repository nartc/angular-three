## NgtPiPipe

This pipe multiples a value with `Math.PI`

```html
<ngt-mesh [rotation]="[-0.5 | pi, 0, 0]">
  <ngt-plane-geometry></ngt-plane-geometry>
</ngt-mesh>
```

## NgtRadianPipe

This pipe converts a value degree to radian

```html
<ngt-mesh [rotation]="[-90 | radian, 0, 0]">
  <ngt-plane-geometry></ngt-plane-geometry>
</ngt-mesh>
```

## NgtSidePipe

This pipe applies `THREE.DoubleSide`, `THREE.FrontSide`, or `THREE.BackSide`

```html
<ngt-mesh>
  <ngt-plane-geometry></ngt-plane-geometry>
  <ngt-mesh-basic-material [side]="'double' | side "></ngt-mesh-basic-material>
</ngt-mesh>
```

## NgtMathPipe

This pipe acts as a helper to use the `Math` object in the template. When working with 3D scene, more often than not you will need to use the `Math` object to calculate angles and such for objects.

```html
<ngt-mesh [rotation]="[1 | math:'sin', 0, 0]">
  <ngt-plane-geometry></ngt-plane-geometry>
</ngt-mesh>
```

:::tip

The value passed in the pipe will be used as arguments to the `Math` function. Eg: `1 | math:'sin'` will return `Math.sin(1)`

:::
