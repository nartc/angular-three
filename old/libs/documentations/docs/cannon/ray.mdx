To interact with [Cannon Ray API](https://pmndrs.github.io/cannon-es/docs/classes/Ray.html), we use `NgtPhysicRaycast` from `@angular-three/cannon`

## Ray Creators

| name              | description                              |
| ----------------- | ---------------------------------------- |
| useRaycastClosest | Create a `Ray` with `RAY_MODE="CLOSEST"` |
| useRaycastAny     | Create a `Ray` with `RAY_MODE="ANY"`     |
| useRaycastAll     | Create a `Ray` with `RAY_MODE="ALL"`     |
