To interact with [Cannon RaycastVehicle API](https://pmndrs.github.io/cannon-es/docs/classes/RaycastVehicle.html), we use `NgtPhysicRaycastVehicle` from `@angular-three/cannon`

## RaycastVehicle Creator

| name              | description                                                                                                                               |
| ----------------- | ----------------------------------------------------------------------------------------------------------------------------------------- |
| useRaycastVehicle | Create a `RaycastVehicle` which is a vehicle helper class that casts rays from the wheel positions towards the ground and applies forces. |

`useRaycastVehicle` returns a `NgtPhysicRaycastVehicleReturn`

```ts
export interface NgtPhysicRaycastVehicleReturn {
  ref: Ref<THREE.Object3D>;
  api: NgtPhysicRaycastVehiclePublicApi;
}

export interface NgtPhysicRaycastVehiclePublicApi {
  applyEngineForce: (value: number, wheelIndex: number) => void;
  setBrake: (brake: number, wheelIndex: number) => void;
  setSteeringValue: (value: number, wheelIndex: number) => void;
  sliding: {
    subscribe: (callback: (sliding: boolean) => void) => void;
  };
}
```
